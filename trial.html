Competetive Programming

Competitive Programming is a mental sport which enables you to code a given problem under provided constraints.

Introduction:

It teaches you how to think?. If you are a programmer you might have understood the deep meaning of these lines quoted by Steve Jobs and you might have also experienced that even after shutting your computer you keep on thinking about programming stuff or code you have written in your project. Once you enter in programming you just don’t learn how to code but you also learn the “art of thinking”, by breaking your code into smaller chunks and then using your logic-based creativity to solve a problem from different angles.
Programming is a challenging role and once you enter this field you will encounter new challenges and you may have to solve some problems which no one has solved before or their solution doesn’t exist anywhere. At that time you are expected to come up with a solution in the least possible time using your problem-solving and logical ability. So the one and clear goal behind these competitive programming is “To prepare a programmer such that his/her logical ability increases and he/she is able to write code for the challenging situation.”

Level 0:

  • Learn a language(preferrable C++/Java)
  • Learn some basic input/output methods Click Here!
  • Start solving 800 Rated problems from Codeforces
  • Do about 30-40 800-1000 Rated Questions (You can always do more!!)
  • Here is a curated list of easy problems
  • Start giving Contests(Sit for the whole duration and think of solution)
  • Boom You are Eligible to proceed to Level 1
  • Tip:Dont just focus on rating(Anyway it'll increase initially)

Level 1:

  • Start giving contests as much as possible and on as many platforms as possible
  • Platforms I prefer are Codeforces!   Codechef!
  • Practice 900-1100 rated problem
  • You should take care that you solve A Question of Codeforces under 20 mins
  • If you are able to solve A during contests ,then you must try to upsolve B after contest
  • Try for 30-35 min ,then go for editorial

Level 2:

  • Start focussing on accuracy and efficiency
  • Try to solve A and B of Codeforces Div-2 ASAP
  • Basically A and B of CF Div-2 is Implementation and simple maths
  • If you are able to solve A and B in 30-40 mins during contest the you are for Next Level
